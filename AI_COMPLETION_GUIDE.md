# AI 补全演员信息功能使用说明

## 🎉 新功能

在演员编辑窗口中添加了 **"AI 补全信息"** 按钮，可以通过千问 AI 自动补全演员的详细信息。

---

## 📍 功能位置

打开 Jvedio 软件：
1. 进入 **演员管理** 界面
2. 点击任意演员进行编辑
3. 在编辑窗口底部，"保存"按钮旁边会出现 **"AI 补全信息"** 按钮

---

## 🚀 使用步骤

### 步骤 1：配置千问 API

确保在 `.env` 文件中配置了千问 API Key：

```env
DASHSCOPE_API_KEY = "sk-你的API密钥"
DASHSCOPE_API_BASE = "https://dashscope.aliyuncs.com/compatible-mode/v1"
```

### 步骤 2：输入演员名字

在演员编辑窗口中，在左侧的搜索框中输入演员名字（如果名字已显示则跳过此步）。

### 步骤 3：点击 AI 补全按钮

点击 **"AI 补全信息"** 按钮。

### 步骤 4：等待 AI 处理

按钮会显示 "正在获取..."，等待几秒钟。

### 步骤 5：查看补全结果

AI 会自动填充以下信息：
- ✅ 生日（Birthday）
- ✅ 年龄（Age）
- ✅ 血型（BloodType）
- ✅ 身高（Height）
- ✅ 体重（Weight）
- ✅ 罩杯（Cup）
- ✅ 胸围（Chest）
- ✅ 腰围（Waist）
- ✅ 臀围（Hipline）
- ✅ 出生地（BirthPlace）
- ✅ 爱好（Hobby）

### 步骤 6：保存信息

点击 **"保存"** 按钮保存更新后的演员信息。

---

## 💡 功能特点

### 1. 智能判断
- 仅补全缺失或为空的字段
- 已有信息不会被覆盖
- 自动验证数据有效性

### 2. 高准确率
- 提示词限定为"日本AV行业演员"
- 提高信息准确度

### 3. 用户友好
- 点击即可获取，无需手动输入
- 按钮状态提示（禁用防止重复点击）
- 成功/失败消息提示

### 4. 置信度显示
- 显示 AI 的置信度百分比
- 帮助判断信息可靠性

---

## ⚠️ 注意事项

### 1. API Key 配置
- 必须在 `.env` 文件中配置有效的千问 API Key
- 如果未配置，点击按钮会提示错误

### 2. 网络连接
- 需要联网才能使用 AI 功能
- 确保可以访问阿里云 DashScope API

### 3. 演员名字
- 演员名字必须准确
- 建议使用日文名或常用别名
- 如果名字错误，AI 可能无法识别

### 4. 数据准确性
- AI 补全的信息仅供参考
- 建议人工核对后再保存
- 如有错误可手动修改

### 5. API 费用
- 每次调用会消耗千问 API 的 Token
- 建议仅在需要时使用

## 🔧 故障排除

### 问题 1：按钮点击无反应

**可能原因**：
- 没有输入演员名字
- 未配置 API Key

**解决**：
- 确保演员名字输入框中有内容
- 检查 `.env` 文件是否配置了 DASHSCOPE_API_KEY

### 问题 2：提示"未配置千问 API"

**解决**：
1. 用记事本打开 `.env` 文件
2. 确保格式正确：
   ```env
   DASHSCOPE_API_KEY = "sk-你的密钥"
   ```
3. 确保 Key 两边有引号

### 问题 3：AI 补全失败

**可能原因**：
- 网络连接问题
- API Key 无效
- 演员名字无法识别

**解决**：
1. 检查网络连接
2. 验证 API Key 是否有效
3. 尝试使用更准确的演员名字

### 问题 4：某些字段没有补全

**可能原因**：
- AI 认为这些信息不确定
- 演员信息公开度不高

**解决**：
- 可以手动补充这些字段
- 这是正常现象，不是错误

## 📝 获取千问 API Key

如果你还没有 API Key：

1. 访问：https://dashscope.console.aliyun.com/apiKey
2. 登录阿里云账号
3. 点击 **创建 API Key**
4. 复制生成的 Key（格式：`sk-xxxxxx`）
5. 填入 `.env` 文件

## 🎯 使用建议

1. 先输入准确的演员名字
2. 点击 AI 补全按钮
